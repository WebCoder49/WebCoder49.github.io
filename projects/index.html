Guess the HEX Colour

A Fun Python Game

This may seem like a trivia game, but it requires some skill. If you want to become a web developer, HEX colour codes are very useful!

To play, you need to open the game and also have a color picker open. Color pickers can be found online - just search the web! You will need to find the HEX code for your colour on the color picker and type it in the game. An example of a HEX code is #000000.
